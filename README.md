# Sales Analytics Dashboard (Excel)

## Project Overview
Built an interactive sales dashboard in Excel to analyze sales performance, customer behavior, product performance, and shipping modes.

## Tools Used
- Excel
- Pivot Tables
- Pivot Charts
- Slicers
- GETPIVOTDATA
- Conditional Formatting

## Dashboard Features
- Total Sales KPIs
- Sales Growth Analysis
- Revenue by Region
- Top Customers
- Top Products
- Shipping Mode Distribution

## Key Insights
- West region generated the highest revenue.
- Consumer segment contributed over 50% of sales.
- Overall sales growth exceeded 50%.
